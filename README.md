# 🌐 Personal Portfolio — Anshuman Kharbade

A modern, fully responsive developer portfolio showcasing my skills, projects, and experience with a **dark glassmorphic design**, **cinematic animations**, and **smooth section navigation**.

## 🚀 Features
- ⚡ Built with **React + Tailwind CSS**
- 🎬 Smooth animations using **Framer Motion**
- 📱 Fully responsive — mobile, tablet & desktop
- 🔗 Section-based navbar with active link scroll
- 🔥 Neon hover effects & interactive UI
- 🧩 Modular and reusable components
- 📬 Contact section with working form (demo)
- 🌑 Modern **dark theme with blur effects**

## 🛠️ Tech Stack
| Frontend | Styling | Animation | Icons |
|----------|---------|-----------|--------|
| React    | Tailwind CSS | Framer Motion | Lucide Icons |

## 📸 Pages Included
